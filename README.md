# Traditional Machine Learning Algorithms

This repository contains practical implementations of Traditional Machine Learning algorithms using Python and Scikit-learn.

The main goal of this repository is to understand how traditional machine learning algorithms work, learn the complete machine learning workflow, and compare model performance before and after applying data preprocessing techniques.

---

## Repository Structure

Each algorithm is implemented in two versions:

### 1. Before Data Preprocessing
Implementation of the algorithm using the raw dataset without applying:
- Data Cleaning
- Missing Value Handling
- Feature Engineering
- Advanced Data Preprocessing

This version is intended for learning the algorithm itself.

### 2. After Data Preprocessing
Implementation of the same algorithm after applying:
- Data Cleaning
- Data Preprocessing
- Feature Encoding
- Feature Scaling
- Feature Engineering (when needed)

This version demonstrates how preprocessing improves model performance.
