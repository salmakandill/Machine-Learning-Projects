## 📌 Project Overview

This project predicts the overall impact of social media on students using a Machine Learning classification model.

The target variable contains three classes:

- Negative
- Neutral
- Positive

The model was built using Logistic Regression after preprocessing and visualizing the dataset.

---


## 🧹 Data Preprocessing

The dataset was prepared by:

- Encoding categorical variables using Mapping
- Applying One-Hot Encoding
- Removing unnecessary columns
- Splitting features and target
- Standardizing numerical features using StandardScaler

---

## 📈 Data Visualization

Created several visualizations to better understand the dataset:

- Gender Distribution
- Daily Social Media Usage Distribution
- Overall Impact Distribution
- Mental Health Score vs Overall Impact
- Confusion Matrix

---

## 🤖 Machine Learning Model

Algorithm:

- Logistic Regression

Workflow:

1. Data Preprocessing
